# 🎯 AI Interview Analyzer

An AI-powered mock interview platform that helps users prepare for technical interviews through resume analysis, AI-generated interview questions, speech-to-text transcription, intelligent answer evaluation, and detailed performance reports.

---

## 📌 Overview

AI Interview Analyzer simulates a real interview experience by analyzing a candidate's resume, generating personalized interview questions using Gemini AI, evaluating spoken responses, and providing actionable feedback with a downloadable PDF report.

This project demonstrates the integration of Artificial Intelligence, Natural Language Processing, Speech Recognition, and Full-Stack Web Development into a practical interview preparation platform.

---

## ✨ Features

### 📄 Resume Analyzer
- Upload PDF resumes
- ATS Score Calculation
- Skill Extraction
- Resume Strength Analysis
- Improvement Suggestions

### 🤖 AI Question Generation
- Generates interview questions based on extracted skills
- Powered by Google Gemini AI
- Personalized technical interview experience

### 🎤 Speech Recognition
- Record answers using microphone
- Automatic Speech-to-Text conversion
- Live transcript generation

### 🧠 AI Answer Evaluation
- Technical Score
- Communication Score
- Overall Performance Score
- Detailed AI Feedback

### 📊 Report Dashboard
- Performance Summary
- Question-wise Analysis
- Progress Indicators
- Overall Rating

### 📑 PDF Report
- Download interview performance report
- Professional formatted report
- Easy to share and review

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/<your-username>/AI-Interview-Analyzer.git
```

---

## Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

Create a `.env` file inside the backend folder:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run the backend:

```bash
uvicorn main:app --reload
```

Backend URL

```
http://127.0.0.1:8000
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend URL

```
http://localhost:5173

```
